# Adidas-US-Sales-Dashboard
# Project Description -->
This project showcases an interactive Excel dashboard built to analyze Adidas US Sales data. The objective of this dashboard is to provide clear insights into sales performance, revenue trends, product-wise contribution, and regional performance across the United States.
